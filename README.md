
# Pong

## Description

This is a recreation of pong

## Controls

 * q and a are up and down repectively for left paddle
 * p and l are up and down repectively for right paddle

## Usage

For Mac or Linux
    python3 pong_v3.py

For Windows
    py pong_v3.py
